¿Que es la programacion competitiva?

La programación es el proceso de crear instrucciones detalladas que una computadora puede seguir para realizar una tarea específica. Es como escribir una receta, pero en lugar de cocinar, estás construyendo un software.

Ejemplo sencillo:

Imagina que quieres crear un programa para sumar dos números. En un lenguaje de programación como Python, podrías escribir:

numero1 = 5;
numero2 = 3;
suma = numero1 + numero2;
print(suma);

Este pequeño código le indica a la computadora que guarde los números 5 y 3 en variables, los sume y luego muestre el resultado en la pantalla.
